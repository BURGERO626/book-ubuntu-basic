dpkg --unpack 的使用

測試環境

以下測試是在「Xubuntu 14.04 64位元」。

使用範例

「dpkg --unpack」的使用範例，請參考『關於「dpkg --unpack」和「dpkg --configure」』這篇。

手冊說明

$ man dpkg

可以看到一段說明

ACTIONS

  ...略...

       --unpack package-file...
              Unpack the package, but don't configure it. If --recursive or -R option is  specified,  package-file  must  refer  to  a
              directory instead.
