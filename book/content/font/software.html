Ubuntu Basic Skill

Ubuntu環境下，一些軟體設定字型的方式

設定Xubuntu 14.04(xfce)的顯示字型

執行「Settings Manager」，找到「Paersonal/Appearance」執行。 選擇「Fonts」這個頁籤，有一個「Default Font」的「Label」，下面有一個「Button」可以按， 就可以選擇你要設定的字型。選好，按下「OK」後，畫面馬上就會有變化了。例如我選「Andika」。

但是你會發現，程式的「Title Bar」的字型，沒有跟著變，因為這個是屬於「Window Manager」管轄的範圍，所以要在另一個地方設定。一樣執行「Settings Manager」，找到「Paersonal/Window Manager」執行。預設在「Style」這個頁籤，就會看到「Title font」這個「Label」，下面一樣有一個「Button」可以按，跟上面的操作模式是一樣的。

會提到這個，是為了「另一篇」而寫的，因為設定的策略，會設定英文字型。通常英文字型，沒有中文。 所以會根據系統的設定順位，系統會去找到相關的中文字型。這樣就可以英文使用一個字型，中文使用一個另一個字型。

Firefox

在「Edit / Preference / Content」可以設定字型。

在「Fonts & Colors / Default font」右方有一個按鈕「Advanced」。按下後，就可以根據不同的語系，設定預設的自行 。還可以根據，「serif」「sans-serif」「monospace」這三個不同的「fontfamily」去設定。

這部份也是要針對「另一篇」而寫的。
