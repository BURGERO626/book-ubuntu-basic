Ubuntu Basic Skill

apt-get 的使用

測試環境

以下測試是在「Xubuntu 14.04 64位元」。

前提

以下以「vim」為範例。

install

安裝「Vim」這個「Package(套件)」。

$ sudo apt-get install vim

有時候直接下載「Deb檔」，然後使用「sudo dpkg -i」的方式來安裝， 會發生有些相依性的套件沒有安裝，這時候只要執行下面的指令， 就會把缺少的相依套件，安裝上去了。

$ sudo apt-get install -f

download

下載「Vim」這個「Binary Package」。也就是下載「Deb檔」，不安裝。

$ apt-get download vim

解開「Deb檔」

$ dpkg -x vim_2%3a7.4.052-1ubuntu3_amd64.deb vim
$ tree vim

$ dpkg -e vim_2%3a7.4.052-1ubuntu3_amd64.deb
$ cat DEBIAN/control

$ ar xf vim_2%3a7.4.052-1ubuntu3_amd64.deb
$ vim control.tar.gz

source

下載「Vim」這個「Source Package」。

$ apt-get source vim

觀看「control」檔

$ cat vim-7.4.052/debian/control
